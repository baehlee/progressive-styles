# SEO Meta — Master Sheet (all pages)

**What this is:** the per-page SEO settings to enter in Lofty's **per-page SEO panel**
for every page, in one place. Work down the list. Titles are kept ≤ ~60 characters,
descriptions ≤ ~160. Focus keyword is the primary target; set it as the page's focus
keyword where Lofty allows. Slugs marked "(Areas tool)" are assigned by Lofty's Areas
tool — match as closely as possible.

> On-page checklist for EVERY page (do once per page): meta title, meta description,
> focus keyword, clean URL slug, alt text on images, links to 3+ other site pages,
> and at least one outbound authority link where it fits.

## E · Site-wide SEO Setup (do once, in the CMS Overview "SEO Setup & Checklist" panel)

From `content/seo/strategy.md §3` — the foundation that has to be right or nothing else matters:

- [ ] **Site title / brand name** set globally (e.g. "Progressive Real Estate Group | Twin Cities")
